# Recap Bot

Telegram-бот делает короткий живой пересказ русскоязычного группового чата. Он хранит сообщения последних 24 часов в памяти, умеет начинать рекап с сообщения, на которое ответили командой `/recap`, расшифровывает голосовые сообщения и кружочки, а затем обращается к OpenAI-совместимому API.

## Возможности

- `/recap` — пересказ доступной переписки за последние сутки.
- `/recap` ответом на сообщение — пересказ начиная с этого сообщения.
- `/help` — краткая инструкция.
- Расшифровка голосовых сообщений и видеокружков.
- Безопасная проверка созданных моделью ссылок на сообщения.

История хранится только в оперативной памяти и пропадает после перезапуска. Для чтения обычных сообщений в группах отключите privacy mode через BotFather, а затем заново добавьте бота в группу.

## Установка и запуск

Требуется Python 3.10 или новее.

```bash
cd recap
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
```

Задайте переменные окружения:

- `TELEGRAM_BOT_TOKEN` — токен Telegram-бота, обязателен.
- `OPENAI_TOKEN` — ключ OpenAI-совместимого API, обязателен.
- `RECAP_LLM_URL` — URL API, по умолчанию `https://api.openai.com/v1`.
- `RECAP_OPENAI_MODEL` — модель рекапа, по умолчанию `google/gemini-2.5-flash-lite`.
- `RECAP_TRANSCRIPTION_MODEL` — модель распознавания, по умолчанию `openai/whisper-large-v3`.
- `RECAP_TRANSCRIPTION_LANGUAGE` — язык распознавания, например `ru`.

Из корня репозитория:

```bash
python -m recap
```

## Тесты

```bash
pytest recap/tests
```

Тесты не обращаются к Telegram или LLM API.
